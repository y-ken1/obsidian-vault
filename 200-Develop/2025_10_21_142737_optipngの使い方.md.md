# optipngの使い方

optipng.exeはpngの圧縮ツールです


1. 最適化して上書き

   ```bash
   optipng -o7 image.png
   ```

2. 最適化して別名保存

   ```bash
   optipng -o7 -out image_opt.png image.png
   ```

3. フォルダ内の PNG をすべて最適化

   ```bash
   optipng -o7 -quiet *.png
   ```

### その他

一時的に PATH を追加

- cmd

  ```cmd
  set PATH=C:\path\to\optipng;%PATH%
  ```

- PowerShell

  ```PowerShell
  $env:PATH = "C:\path\to\optipng;" + $env:PATH
  ```
